# Afiliaciones y Traspasos Completo

## Descripción
Gestión avanzada de afiliaciones, control de tipos de contratos de afiliación, traspasos inter-ARS con importación de consumos históricos, y codificación geográfica nacional (regiones, provincias, sectores).

## Entidades Principales
- `AffiliateGroup`: Agrupaciones corporativas para facturación patronal.
- `AffiliateContract`: Contratos específicos por tipo de cotizante.
- `BusinessUnit`: Unidades operativas de atención.
- `GeographicCode`: Padrón geográfico nacional (regiones, provincias, sectores).
- `AffiliateTransaction`: Auditoría en tiempo real de transacciones de afiliación.

## Instrucciones Demo
1. Ingrese a **Afiliaciones -> Gestión de Traspasos**.
2. Registre el traspaso de un afiliado indicando la procedencia histórica y guarde el registro para ver el log de auditoría.
