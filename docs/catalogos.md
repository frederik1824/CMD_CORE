# Catálogos Centralizados del Sistema

## Descripción
Centralización de todas las tablas de equivalencias y catálogos generales del sistema, permitiendo habilitar o inhabilitar códigos de estados, parentescos, tipos de afiliación y más.

## Entidades Principales
- `Catalogo`: Tabla unificada de catálogos generales del Core.

## Instrucciones Demo
1. Ingrese a **Administración -> Catálogos Centralizados**.
2. Vaya a ver los elementos de cualquier grupo de catálogo y cambie el estatus a inactivo.
