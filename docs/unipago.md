# Simulador de Integración Unipago

## Descripción
Consola avanzada del simulador de Web Services de Unipago, permitiendo auditar logs de peticiones API y revisar notificaciones del SUIR de cartera y coberturas.

## Instrucciones Demo
1. Acceda a **Simulador Unipago -> Simulador Avanzado V2** en el menú.
2. Audite las llamadas en **Consulta de Procesos Unipago** para revisar el log de peticiones API simuladas.
